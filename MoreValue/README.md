# More Value

## Description
This mod gives a minor increase to the value of items to help obtain more upgrades in the beginning
(This mod is reccommend more for playing with friends)

## Features
- Current item values are mulitpled by 2x
- Original item values are multipled by 2x

## Installation
1. Make sure BepInEx is installed in your REPO folder
2. Download the mod from Thunderstore or git hub
3. Extract the files to your 'REPO/BepInEx/Plugins' folder (Thunderstore should do this for you)
4. Launch the game

## Configuration
Currently no configuration options
- Dollar Value Current: 2x
- Dollar Value Original: 2x
